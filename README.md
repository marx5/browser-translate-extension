## üá∫üá∏ English

**Browser Translation Extension** is a powerful tool designed for seamless translation directly on webpages or via a popup interface. It supports top-tier services like Google Translate, Gemini AI, OpenAI (ChatGPT), and MyMemory.

### ‚ú® Key Features
*   **Inline Translation:**
    *   Highlight text -> Translation icon appears.
    *   Click the icon to view translation instantly.
    *   Icon **auto-hides** after 5 seconds of inactivity.
*   **Popup Interface:**
    *   Modern UI with Dark Mode support.
    *   **Redesigned Settings:** Dedicated settings view with intuitive Home/Settings toggle.
*   **Multi-Service Support:**
    *   **Google Translate:** Free and fast (Default).
    *   **Gemini AI & OpenAI:** Context-aware AI translation (Requires API Key).
    *   **MyMemory:** Collaborative translation archive.
*   **Other Features:**
    *   **Text-to-Speech:** Listen to original and translated text.
    *   **Phonetics (IPA):** Displays IPA pronunciation for English.
    *   **Secure Key Management:** Enter API Keys directly in the Settings UI (securely stored in browser), no code editing required.

### üöÄ Installation
1.  Download or Clone this repository.
2.  Open your browser (Chrome/Edge/Brave) and go to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked** and select the extension folder.

### ‚öôÔ∏è API Configuration
To use **Gemini** or **OpenAI**:
1.  Click the extension icon to open the Popup.
2.  Click the **Settings** button (‚öôÔ∏è icon).
3.  Enter your API Key into the corresponding field.
4.  Click **Save Settings**. You're set!
